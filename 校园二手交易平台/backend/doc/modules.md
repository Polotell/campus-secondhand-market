## 二、项目开发流程（6 周计划）

```
第1周 ─── 需求分析 + 系统架构设计 + 数据库设计（E-R + 建表）
第2周 ─── 技术选型 + 工程骨架搭建 + 接口文档定稿 + 鉴权登录
第3周 ─── 用户模块 + 商品模块 + 管理员审核（CRUD + 审核流）
第4周 ─── 购物车 + 下单 + 资金托管 + 定时结算（核心难点）
第5周 ─── 退货 + 评价 + 钱包积分 + 可选功能（轮播/封禁/黑名单）
第6周 ─── 联调测试 + 边界场景回归 + 报告撰写 + 答辩准备
```
